## Installation
You only need to have node.js and bower installed. 

Installing node.js: 

See http://nodejs.org/download/


Install bower: 
```sh
$ npm install -g bower
```
Install node modules:
```sh
$ npm install
```

Install frontend dependencies: 
```sh
$ bower install
```
